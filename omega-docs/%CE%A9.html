(static) frag(content, cloneElsopt) → {DocumentFragment}

Creates a DocumentFragment containing the given content.
Parameters:
Name Type Attributes Default Description
content String | HTMLTemplateElement | Element The content to insert into the new fragment.
cloneEls Boolean <optional>
 false (Only applies if the content parameter is an HTMLElement). If true, the contents of the given element will be cloned. If false, the contents will be adopted. (If content is an HTMLTemplateElement, the contents will always be cloned.)
Source:
  • omega.js, line 170
Returns:
The content of the fragment will vary depending on the type of the content parameter:
undefined
The returned fragment will contain no content.
String
The returned fragment will contain all nodes resulting from parsing content as HTML. This should be assumed to be an unsafe parse - i.e., do not pass untrusted HTML to this method.
HTMLTemplateElement
The returned fragment will contain a single child, a clone of the template's content attribute.
Element
For an Element that is not an HTMLTemplateElement, the returned fragment will contain a single child, the passed-in content. If clone is true, this will have been cloned via cloneNode. Otherwise, the given element will be adopted.

DocumentFragment
